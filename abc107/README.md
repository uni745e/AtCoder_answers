## C
とり得るパターンは  

- 正方向に向かってi本のロウソクに火をつけた後原点に戻り、負方向に向かってj本のロウソクに火をつける(cost_rl)
- 負方向に向かってj本のロウソクに火をつけた後原点に戻り、正方向に向かってi本のロウソクに火をつける(cost_lr)

の2パターンしかない。  
この時, `j = K - i`, `0 <= i, j <= K`

0 <= i <= Kの範囲で全探索して、最小値`min(ans, cost_rl, cost_lr)`を求める

今回の問題は一直線上にロウソクが並んでいたが、似たような問題として[Static Sushi (ABC095 D問題)](https://beta.atcoder.jp/contests/abc095/tasks/arc096_b)がある。こちらは一直線ではなく円周上にオブジェクトが並んでいる。  
これ解いてたおかげでACできた。
