# AtCoder_answers  
AtCoderコンテストのコードをアップするリポジトリ  
元々Gistで管理していたが、使った手法や気づきを一緒に管理したかったのでこちらで管理することにした  
過去のコードも少しづつアップする予定

```
├── irregular_contest：企業が開催したコンテスト
├── algo：蟻本で学んだことをまとめる
│   ├── graph：グラフ探索系
│   └── greedy：貪欲（この分け方は良くないかも）
└── regular_contest：ABC, ARC, AGCコンテスト
```
